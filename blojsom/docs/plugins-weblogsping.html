description

This plugin sends a ping to weblogs.com after new blog entries have been detected. The ping is sent via XML-RPC according to the weblogs.com ping XML-RPC specification. On server startup, the plugin uses the current date and time to determine whether new entries have been added. If a new entry has been detected, the plugin will set its check to that entry's date and time.

installation and setup

In your plugin.properties file, add the following line: 

    weblogs-ping=org.ignition.blojsom.plugin.weblogsping.WeblogsPingPlugin
You may then add "weblogs-ping" to one of the flavor-based plugin chains. For example: 
    html.blojsom-plugin-chain=weblogs-ping
