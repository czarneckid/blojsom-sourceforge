description

The simple search plugin looks for a user's query in either the blog entry title or blog entry description.

installation and setup

In your plugin.properties file, add the following line: 

    simple-search=org.ignition.blojsom.plugin.search.SimpleSearchPlugin
You may then add "simple-search" to one of the flavor-based plugin chains. For example: 
    html.blojsom-plugin-chain=simple-search

usage and URL parameters

URL parameters

The following table describes the URL parameters that affect the output of the simple search plugin.

URL parameter Accepted values Usage
query Contains the query text to look for in the blog entry title or blog entry description. /blog/?query=blojsom
