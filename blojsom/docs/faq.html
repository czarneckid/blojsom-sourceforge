general

Q: What does blojsom stand for?
A: blojsom is an acronym for (blo)gging in (j)ava ... (so)mewhat ... (m)aybe. I was looking for a way to play off the name, blosxom, blojsom's not-so-distant "relative", and blojsom stuck.

Q: How do you pronounce blojsom?
A: Phonetically, I guess it would be bloj-sum.

Q: Is it Blojsom or blojsom?
A: blojsom.

Q: How do I create my first blog entry?
A: blojsom renders your blog based on what files it finds in the directory you set as your blog-home (see installation). In this directory, each individual file is treated as a seperate blog entry. You can control which files are processed and which are ignored using the blog-file-extentions and blog-directory-filter entries in /WEB-INF/blojsom.properties.

Q: How do I create different blog categories?
A: All you have to do is simply create new directories underneath your blog-home directory and blojsom will automatically recognize them as categories. You can further control the categories using optional meta-data, an example is a more descriptive name for the category. See customization for more details.

Q: Does blojsom require a database?
A: No! This is contrary to the original zen and philosophy of blojsom's "relative" blosxom. blojsom uses the file system (folders and files) as its content database.

moblogging

Q: I hear a lot about moblogging ... is it supported in any way by blojsom?
A: Yes. Well, there are different ways to support moblogging and with blojsom, you can use blojsim, an instant messenger add-on for blojsom to blog via AIM, Jabber, or MSN.

Q: Besides blojsim, is there any other support for moblogging?
A: Eventually, we will get around to writing an e-mail interface so that you can send e-mail to a mailbox that blojsom will monitor and populate your blojsom blog with the content.

xml-rpc

Q: Can I edit my blojsom blog via XML-RPC?
A: Yes. Starting with blojsom 1.4, blojsom will include an XML-RPC servlet and XML-RPC handlers for the Blogger API. We also intend to support the MetaWeblog API as well as the Live Journal API.

extensions / plug-ins

Q: Are there any extensions available for blojsom?
A: Yes. In fact there is one (at the moment). Check out blojsim, which is an instant messenger add-on for blojsom. Using blojsim, you can blog directly to blojsom from your mobile phone or any other IM-capable device.

Q: What plugins are available for blojsom?
A: As blojsom users write plugins for blojsom, we will maintain a list of plugins for blojsom. Starting with blojsom 1.4, there are a number of plugins available using the plugin API.
