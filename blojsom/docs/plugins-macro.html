macro expansion plugin

The macro expansion plugin will extract tokens from a blog entry and replace them with other content. This is a greate timesaver when you can simply enter $HOME$ in your blog entry and macro plugin will convert that to My Blog Home as an example.

managing your macros


The file /WEB-INF/macros.properties contains simple name=value pairs that represent the token and the value to replace it with. It can be simple text expanstion or a full <a href="http://www.news.com">NEWS!</a> URL

When you write your entries, simply surround your macro tokens with the $ symbol. An example is $blogsom$ (tokens are case insensative)
