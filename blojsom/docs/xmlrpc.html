installation and default configuration

Starting with blojsom 1.4, an XML-RPC servlet is included with blojsom that provides support for the Blogger API and the MetaWeblog API.

Edit the contents of /WEB-INF/authorization.properties to setup all the valid username/password pairs to control who can post via XML-RPC. And you're done ;)

To send a message to the service, point your blogger tool or XML-RPC client at your http://www.your.blog.com/xmlrpc

Replace www.your.blog.com with the url for your blog and get blogging!

writing a custom XML-RPC handler

If you are writing a custom XML-RPC handler for blojsom, you must write your class to implement the AbstractBlojsomAPIHandler interface. Take a look at the source code for the BloggerAPIHandler and the MetaWeblogAPIHandler to see how it's done.

registering a custom XML-RPC handler

To register your custom XML-RPC handler with blojsom, you must modify the /WEB-INF/blojsom-xmlrpc.properties file. The name of this file can be configured via the blog-xmlrpc-configuration initialization parameter in your web.xml file.

The contents of the file looks similar to the following: 

    #
# blojsom XML-RPC API Handlers
#
blogger=org.ignition.blojsom.extension.xmlrpc.handlers.BloggerAPIHandler
metaWeblog=org.ignition.blojsom.extension.xmlrpc.handlers.MetaWeblogAPIHandler
You simply register a particular API handler under a given name. In the first case above, the "blogger" API handler is handled by the org.ignition.blojsom.extension.xmlrpc.handlers.BloggerAPIHandler class.
