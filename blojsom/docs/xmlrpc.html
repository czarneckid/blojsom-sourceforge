installation and default configuration

1. Starting with blojsom 1.4, an XML-RPC servlet is included with blojsom that provides support for the Blogger API.
2. Edit the contents of /WEB-INF/authorization.properties to setup all the valid username/password pairs that can post via xml-rpc

And your done ;) By default, blojsom's XML-RPC interface is enabled and setup to work with a subset of the Blogger API.

To send a message to the service, point you blogger tool or XML-RPC client at your http://$BLOG_URL/xmlrpc (replace $BLOG_URL with the url for your blog) and get blojing!
